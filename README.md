# Katapayadi system

```
Katapayadi system (Devanagari: कटपयादि, also known as Paralppēru, Malayalam: പരല്‍പ്പേര്) of numerical notation
is an ancient Indian alphasyllabic numeral system to depict letters to numerals for easy remembrance of numbers
as words or verses.
```
```
Katapayadi is a numerology system that was used in ancient India to encode words into numbers. In this system,
each letter of the Sanskrit alphabet is assigned a numerical value, and words are formed by adding up the values
of their constituent letters
```
